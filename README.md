# UnoRadioFrog

Uses nRF24L01 to communicate two Arduino's. Control LED and Motor with button and Joystick.

## Transmitter
- Trema shield
- Trema Button with LED
- Joystick (x, y, sw)

## Receiver
- Amperka MotorShield
- Trema LED module
- DC Motor
